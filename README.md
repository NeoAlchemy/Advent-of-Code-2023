# Advent-of-Code-2023

## Install Application

- [install pipx](https://github.com/pypa/pipx) (if not available)

- install poetry

pipx install poetry==1.7.1

- install poetry dependencies

    ```bash
        poetry install
    ```

## Run Application

- use the run flask command

    ```bash
        poetry run flask --app src/app/main run
    ```
